# Stock Verse – Smart Supermarket POS System

Stock Verse is a web-based barcode-driven Point of Sale (POS) system developed using Python, Streamlit, and SQLite to automate supermarket operations. The system provides efficient inventory management, real-time barcode scanning, automated billing, and sales analytics.

Key features include dynamic pricing based on predefined business rules, expiry-aware discount generation to reduce product wastage, real-time inventory updates, and email-based digital receipt generation. The application improves billing speed, minimizes manual errors, and enhances inventory control for small and medium-sized retail businesses.

## Technologies Used

* Python
* Streamlit
* SQLite
* Barcode Scanner Integration
* Email Services (SMTP)

## Features

* Barcode-based product identification
* Automated billing and receipt generation
* Real-time inventory tracking
* Dynamic pricing mechanism
* Expiry-based discount offers
* Sales analytics and reporting
* Email receipt delivery

## Benefits

* Faster checkout process
* Reduced manual errors
* Better stock management
* Reduced product wastage
* Improved customer experience
